# OdorSig 480-Pair Dataset

Complete OR–odorant binding affinity dataset generated by the 
OdorSig automated docking pipeline across 40 olfactory receptors 
and 12 odorants (480 pairs total).

## Files
- `odorsig_480_pairs.csv` — main dataset
- `extract_interactions.py` — code to extract H-bond and 
   hydrophobic contacts from docking poses

## Column Descriptions

| Column | Description |
|--------|-------------|
| receptor | Olfactory receptor UniProt ID |
| odorant | Odorant name |
| dG_run1 | Binding energy run 1 (kcal/mol) |
| dG_run2 | Binding energy run 2 (kcal/mol) |
| dG_run3 | Binding energy run 3 (kcal/mol) |
| dG_mean | Mean binding energy across 3 runs (kcal/mol) |
| sigma | Standard deviation across 3 runs (kcal/mol) |
| hbond_count | H-bond count in best docking pose |
| hydrophobic_contacts | Hydrophobic contacts in best pose |
| binding_class | strong / moderate / weak (per Hsin et al.) |
